# 🛒 Flipkart Clone (E-Commerce Web Application)

A modern, responsive e-commerce front-end clone of Flipkart built from scratch using pure HTML5, CSS3, and Vanilla JavaScript.

---

## 🚀 Live Demo

🔗 **[View Live Flipkart Clone](https://satyam2025maurya-ops.github.io/Flipkart-Clone/)**

---

## ✨ Key Features

- **Header & Search Bar**: Sticky top navigation bar with integrated live product search.
- **Category Banner**: Smooth horizontal-scroll category section for desktop and mobile devices.
- **Product Cards & Grid**: Responsive multi-column layout with smooth hover effects.
- **Live Search Filter**: Real-time JavaScript filter that hides non-matching items as you type.
- **Interactive Shopping Cart**: Dynamic cart counter badge updating in real-time when items are added.
- **Dark Theme Footer**: Full-width footer containing policy, company links, and addresses.
- **Mobile First & Responsive**: Optimized for all screen sizes using CSS Flexbox, Grid, and Media Queries.

---

## 🛠️ Tech Stack

* **HTML5**: Structured semantic web pages
* **CSS3**: Layouts using Flexbox, CSS Grid, and responsive breakpoints
* **JavaScript (ES6+)**: DOM manipulation and real-time event listeners
